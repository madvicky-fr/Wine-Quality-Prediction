# Wine Quality Prediction using Machine Learning

## Project Overview

This project predicts whether a wine is GOOD or BAD based on its physicochemical properties.

A wine is classified as:

- GOOD (1) → Quality >= 7
- BAD (0) → Quality < 7

---

## Dataset

Wine Quality Dataset

Features include:

- Fixed Acidity
- Volatile Acidity
- Citric Acid
- Residual Sugar
- Chlorides
- Free Sulfur Dioxide
- Total Sulfur Dioxide
- Density
- pH
- Sulphates
- Alcohol

Target:

quality_label
